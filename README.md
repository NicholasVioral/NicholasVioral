## Projects

**[AI Soccer Clips Compiler](https://github.com/NicholasVioral/AI-Soccer-Clips-Compiler)**

Full-stack highlight generator that syncs broadcast video with StatsBomb event data. Uses OpenCV and Tesseract OCR to extract match time from video frames, maps 3,500+ match events to video timestamps, and compiles clips with FFmpeg. Built with React, Flask, and Python.

**[Algorithm Visualizer](https://github.com/NicholasVioral/Algorithm-Visualizer)**

Interactive educational tool that visualizes 25+ algorithms — sorting, graph traversal, dynamic programming, recursion — with step-by-step execution and variable tracking. Includes an algorithm detection system using 150+ regex patterns to identify algorithms from pasted code.
